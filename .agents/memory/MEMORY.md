- [Drizzle numeric columns return strings](drizzle-numeric-string-mode.md) — Postgres `numeric` columns round-trip as JS strings unless declared with `mode: "number"`, which silently breaks Zod response validation on money fields.
- [Orval non-coercing date query params](orval-zod-date-query-params.md) — Orval emits `zod.date()` (not `.coerce.date()`) for OpenAPI `format: date` query params, rejecting string query values; parse those params manually in the route handler.
- [tsx not preinstalled; tsc project refs use dist types](workspace-tsc-project-refs-seed-scripts.md) — seed/one-off scripts should use plain `.mjs` + `pg` from a package that already depends on it; `tsc --noEmit` with project references reads referenced packages' built `dist/*.d.ts`, so rebuild (`tsc -b`) a lib after changing its schema or downstream typecheck sees stale types.
